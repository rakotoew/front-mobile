# Save Humanity
-- --
Page Web pour afficher les données de l'application mobile Save Humanity

## Installation et Lancement
-- --
Installer les dépendances
```bash
npm install
```

Modifier l'adresse de l'api dans le fichier src/pages/_app.tsx

Il faut ensuite build l'application avec la commande suivante
```bash
npm run build
```

Vous pouvez lancer l'application avec la commande suivante
```bash
npm run start
```
